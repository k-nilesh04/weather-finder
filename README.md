# 🌦️ Weather Finder App

A responsive and user-friendly weather application that provides **real-time weather information** for any city. Simply enter a city name to get the latest weather details, including temperature, humidity, wind speed, and weather conditions.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature updates
- 💧 Displays humidity
- 🌬️ Shows wind speed
- ☁️ Dynamic weather icons
- 📱 Fully responsive design
- ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API

## ⚙️ Installation

1. Clone this repository

```bash
git clone https://github.com/k-nilesh04/weather-finder.git
```

2. Navigate to the project folder

```bash
cd weather-finder
```

3. Open `index.html` in your browser.

---

## 🔑 API Setup

1. Get a free API key from **OpenWeatherMap**.
2. Replace the API key in `app.js`.

```javascript
const API_KEY = "YOUR_API_KEY";
```
